# Programming Sample Programs

This repository contains various sample programs written in **Python** and **Java** along with basic **Data Structures** examples.

The programs are created for learning programming concepts, practicing logic building, and understanding fundamental coding techniques.

## 📂 Files Included
This repository includes programs related to:
- Basic Python programs
- Java practice programs
- Conditional statements
- Input and output
- Operators
- String operations
- Array programs
- Basic Data Structure examples (Stack, Queue, Linked List)

## 🎯 Purpose
- Practice programming concepts
- Improve problem-solving skills
- Learn basic data structures
- Build coding fundamentals

## 💻 Languages Used
- Python
- Java
